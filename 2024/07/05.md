## わかったこと
- CSSでスクロールを禁止するには`overflow: hidden;`, 一部のみスクロール可能にするには`overflow: auto;`を使う
- スクロールバーを非表示にするには`::-webkit-scrollbar`を使う
- 以下は全体ではスクロール禁止だが, mainのみスクロール可にした状態

```css:style.css
html, body {
  overflow: hidden;
}

main {
  overflow: auto;
}

main::-webkit-scrollbar {
  display: none;
}
```
